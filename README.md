# SmokeScreen ENGINE

Full-stack license and auth system with Clerk integration (Next.js App Router), GUI, and ENGINE.exe.

- Website: Next.js with Clerk middleware and provider
- API: `/api/clerk-sync`, `/api/keys/redeem-clerk`, `/api/keylogger`
- Static pages: pricing, sign-in, sign-up, forgot, contact-sales
- GUI: ClerkAuth + MsPingStatus + KeyCache sync
- ENGINE: Key generation + KEYMAKERLOGGER.zxy

Deployed on Vercel.
